# Java 由来

Java 最早是 Sun 公司在 1990 年代为嵌入式设备准备的一门语言，项目代号 **Oak**。随着互联网浪潮爆发，团队意识到“跨平台”才是更具潜力的方向，于是在 1995 年将其正式命名为 Java，并在 Netscape 浏览器中以 Applet 形式亮相。

## 设计理念

1. **一次编写，到处运行**：借助 JVM 屏蔽操作系统差异。
2. **安全**：通过字节码校验、类加载器和安全管理器构建沙箱。
3. **简洁**：去掉了 C/C++ 中复杂的指针和多继承语法。

## 当下意义

虽然 Applet 早已成为历史，但 Java“跨平台 + 稳定可靠”的 DNA 被保留下来，广泛应用于企业级后端、大数据、移动端乃至云原生场景。
