小册简介

本章节给出整个 Java 学习小册的全貌，帮助你弄清楚“为什么学、学什么、怎么学”。

学习路径

  1. 前言：了解 Java 的诞生与发展。
  2. 环境搭建：安装 JDK、IDEA，熟悉命令行工具。
  3. 语法基础：变量、流程控制、方法。
  4. 核心能力：面向对象、集合、异常、IO。
  5. 进阶模块：并发、网络、常用工具、新特性。

阅读建议

  • 左侧目录按照章节递进，可作为线性课程。
  • 每篇文章末尾会给出练习和延伸阅读。
  • 配合官方文档与开源项目，才能真正掌握。
